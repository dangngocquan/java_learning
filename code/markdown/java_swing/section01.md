**<h2 id="01">Displaying Text in a Dialog Box</h2>**

### ***Code & Output***
```
import javax.swing.JOptionPane;

public class Dialog1 {
    public static void main(String[] args) {
        JOptionPane.showMessageDialog(null, "Hello\nBris\nToday is 11/11/2022");
    }
}
```
